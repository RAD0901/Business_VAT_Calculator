üìã File Requirements

Required Columns:

  • TaxCode - The tax code (see valid codes below)
  • TaxDescription - Description of the tax transaction
  • TrCode - Transaction code
  • TaxRate - VAT rate as percentage (e.g., 15)
  • TaxAmount - VAT amount in currency
  • ExclAmount - Amount excluding VAT
  • InclAmount - Amount including VAT

Valid Tax Codes:

Input VAT (VAT you paid):

  • CASH - Cash transactions
  • JC - Journal credit entries
  • JNL - Journal entries
  • SINV - Supplier invoices
  • JD - Journal debit entries
  • RC - Receipt entries

Output VAT (VAT you collected):

  • IS - Invoice sales
  • INV - Invoices
  • RTS - Returns

üìä Understanding Results

The VAT Calculator provides a comprehensive breakdown of your VAT liability:

  • Input VAT Total - Total VAT you paid on purchases
  • Output VAT Total - Total VAT you collected on sales
  • Net VAT Amount - The difference (Output VAT - Input VAT)
  • Detailed Breakdown - Transaction counts and amounts by tax code

Interpreting Net VAT Amount:

  • Positive amount - You owe VAT to SARS
  • Negative amount - SARS owes you a VAT refund
  • Zero amount - No VAT liability or refund

üîß Troubleshooting

Common Issues:

  • File not uploading - Check file format (.xlsx or .xls) and size (<50MB)
  • Invalid tax codes - Ensure tax codes match the valid list above
  • Missing columns - Verify all required columns are present
  • Processing errors - Check for non-numeric values in amount fields

Data Validation:

  • Tax codes must be exactly as listed (case-sensitive)
  • Numeric fields (rates, amounts) must contain valid numbers
  • Empty or invalid rows are automatically skipped
