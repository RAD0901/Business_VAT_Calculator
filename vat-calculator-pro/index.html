Help & Support

Everything you need to know about using VAT Calculator Pro

Getting Started

  1. Prepare your Excel file with the required columns
  2. Navigate to the Upload page
  3. Select or drag your Excel file
  4. Click "Process File" to start calculation
  5. View and export your professional VAT report

Required Columns

  • TaxCode: 1 (Standard 15%), 3 (Zero-rated), or 5 (Exempt)
  • TaxDescription: Description of the tax type
  • TrCode: Transaction code (see codes below)
  • TaxRate: Tax rate percentage
  • TaxAmount: VAT amount in Rands
  • ExclAmount: Amount excluding VAT
  • InclAmount: Amount including VAT

Transaction Codes

Input VAT Codes (Purchases)

  • CASH - Cash purchases
  • JC - Journal credit
  • JNL - Journal entry
  • SINV - Supplier invoice
  • JD - Journal debit
  • RC - Received cash

Output VAT Codes (Sales)

  • IS - Invoice sale
  • INV - Invoice
  • RTS - Return sale

About VAT Calculator Pro

Professional VAT calculation tool for South African businesses

Purpose

VAT Calculator Pro was built to help South African businesses and accountants quickly process Excel-based VAT data into professional, SARS-compliant reports. Our tool eliminates manual calculations and reduces errors while ensuring compliance with South African VAT regulations.

Key Features

  • Client-side processing for maximum security
  • Support for all South African VAT codes and rates
  • Professional PDF and Excel report generation
  • Real-time validation and error checking
  • Mobile-responsive design
  • No registration or software installation required

Privacy & Security

Your data privacy is our top priority. VAT Calculator Pro processes all data locally in your browser - no files are uploaded to any server. Your sensitive VAT information never leaves your computer, ensuring complete confidentiality and compliance with data protection requirements.

Technical Specifications

  • Supports Excel files up to 50MB
  • Processes up to 10,000+ transactions in under 5 seconds
  • Compatible with all modern browsers
  • Works offline after initial load
  • No server dependencies
